Technologies Used

  • Python
  • TensorFlow
  • OpenAI Gym
  • Pygame

    • Python: The primary programming language used for developing the entire program.

  • Pygame: Used for creating the game environment, handling graphics rendering, user input, and game loop for the Snake game.
  • PyTorch: Used for implementing the deep learning model and training process.
  • Key modules from PyTorch: torch, The core PyTorch library. torch.nn, Contains modules and classes for building neural networks. torch.optim, Contains optimization algorithms like Adam. torch.nn.functional, Provides functions for different operations like activation functions.
  • Matplotlib: Used for plotting and visualizing the training progress. Uses matplotlib.pyplot, Provides functions for creating plots and visualizations.
  • IPython: Used for dynamically updating plots in Jupyter notebooks. Uses IPython.display to provide functions for clearing and displaying output in Jupyter notebooks.
  • Numpy: Used for numerical operations and handling arrays. Key module uses numpy to provide support for large, multi-dimensional arrays and matrices, along with a collection of mathematical functions to operate on these arrays. OS is used for interacting with the operating system, particularly for checking the existence of directories and files, and creating directories.
  • Collections: Used for data structures like deque and namedtuple. Key modules: collections.deque, provides a double-ended queue for efficient memory management. collections.namedtuple, provides a way to create tuple subclasses with named fields.
  • Enum: Used for creating enumerations, which are a set of symbolic names bound to unique, constant values. Key module: enum.Enum, Base class for creating enumerations.
