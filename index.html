AI Plays Snake

Project Overview

This project involves developing an artificial intelligence (AI) agent that can play the classic Snake game. The AI uses machine learning techniques to improve its performance and achieve high scores.

Goals

  • Develop an AI capable of playing Snake autonomously.
  • Implement machine learning algorithms to enhance the AI's performance.
  • Analyze the AI's decision-making process.

Technologies Used

  • Python
  • TensorFlow
  • OpenAI Gym
  • Pygame

How It Works

The AI is trained using a reinforcement learning approach. It interacts with the Snake game environment, receiving rewards for desirable actions (like eating food) and penalties for undesirable actions (like hitting the wall or itself). Over time, the AI learns to make better decisions to maximize its score.

Results

Through training, the AI has been able to achieve high scores consistently. The learning process is visualized using graphs and game play videos, showing the improvement in the AI's performance over time.

Future Work

Future enhancements could include:

  • Improving the AI's learning speed and efficiency.
  • Testing the AI on different variations of the Snake game.
  • Exploring other game environments and applying similar AI techniques.
