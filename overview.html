Project Overview

This project involves developing an artificial intelligence (AI) agent that can play the classic Snake game. The AI uses machine learning techniques to improve its performance and achieve high scores.

Simple game where a user navigates a snake to eat apples.

Snake grows with each apple bite.

Ultimate objective is to eat apples and grow without dying (eating itself or hitting walls).
